Политика конфиденциальности

Настоящая Политика конфиденциальности описывает порядок обработки информации, получаемой в рамках использования веб-сайта TrainGain.

1. Общие положения

Сайт TrainGain является учебным дипломным проектом, разработанным в рамках выполнения выпускной квалификационной работы. Сайт не является коммерческим сервисом, а его разработчики не выступают в статусе юридического лица или индивидуального предпринимателя.

2. Какие данные могут обрабатываться

При использовании сайта могут обрабатываться следующие данные:

  • логин пользователя;
  • адрес электронной почты;
  • пароль в зашифрованном виде (в виде хэша, без хранения в открытом виде);
  • данные, вводимые пользователем в разделах сайта, например параметры для расчёта питания;
  • технические данные, автоматически возникающие при работе сайта.

3. Цели обработки данных

Данные пользователей обрабатываются исключительно для следующих целей:

  • регистрация и авторизация на сайте;
  • восстановление доступа к аккаунту;
  • подтверждение адреса электронной почты;
  • сохранение пользовательских настроек и результатов в личном кабинете;
  • обеспечение корректной работы функций сайта;
  • повышение безопасности аккаунта.

4. Хранение и защита данных

Пароли пользователей не хранятся в открытом виде. Для защиты учётных данных используется механизм хэширования паролей. Доступ к данным ограничивается средствами программной логики сайта.

Разработчики проекта принимают разумные технические меры для снижения риска несанкционированного доступа к данным, однако не могут гарантировать абсолютную защиту информации в сети Интернет.

5. Передача данных третьим лицам

Сайт не осуществляет продажу, передачу или распространение персональных данных пользователей третьим лицам, за исключением случаев, когда такая передача необходима для работы отдельных технических функций сайта, например отправки сервисных писем на указанный пользователем адрес электронной почты.

6. Использование электронной почты

Адрес электронной почты пользователя может использоваться для:

  • подтверждения регистрации;
  • смены адреса электронной почты;
  • восстановления пароля;
  • других технических уведомлений, связанных с работой сайта.

7. Платежи

На текущем этапе в рамках проекта не предусмотрена обработка онлайн-платежей, банковских карт или платёжных реквизитов пользователей.

8. Права пользователя

Пользователь имеет право:

  • использовать сайт добровольно;
  • не регистрироваться на сайте, если не согласен с условиями обработки данных;
  • обратиться к разработчику проекта по вопросам, связанным с использованием сайта;
  • прекратить использование сайта в любой момент.

9. Изменение политики

Политика конфиденциальности может быть изменена при доработке проекта. Актуальная версия размещается на данной странице.

10. Контактная информация

По вопросам, связанным с работой сайта и обработкой данных, пользователь может связаться с разработчиком по адресу электронной почты, указанному в футере сайта.
